# 射击游戏

这是一个简单的命令行射击游戏，玩家控制飞船，射击从上方飞来的敌机。

## 游戏玩法

- 玩家飞船位于屏幕底部，用 `^` 表示
- 敌机会从屏幕顶部随机位置出现，用 `V` 表示
- 玩家可以发射子弹，用 `|` 表示，击落敌机
- 每击落一个敌机得10分
- 如果敌机到达屏幕底部，玩家会失去一条生命
- 如果敌机与玩家飞船碰撞，玩家也会失去一条生命
- 玩家初始有3条生命

## 控制方式

- `A` - 向左移动
- `D` - 向右移动
- `S` - 发射子弹
- `Q` - 退出游戏

## 游戏特点

- 随着分数增加，敌机生成速度会加快
- 简单易懂的游戏机制
- 经典的射击游戏体验

## 运行游戏

```bash
python shoot_game_final.py
```

注意：在某些环境中，你可能需要使用交互式终端来运行游戏以获得完整的控制体验。
